# Variable in Python
## 1. Comment trong Python
- Có 2 cách để comment trong python:
  - C1: Sử dụng dấu `#` để comment 1 dòng.
  - C2: Sử dụng dấu `"nội dung comment"` ==> ít sử dụng do tùy thuộc vào HĐH hoặc phiên bản.
